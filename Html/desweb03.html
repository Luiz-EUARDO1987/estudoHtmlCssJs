Atividade Teórica – Introdução ao JavaScript

O que é JavaScript?

JavaScript é uma linguagem de programação amplamente utilizada para criar interatividade e dinamismo em páginas web. Ele é uma das tecnologias essenciais no desenvolvimento web, formando, junto com HTML e CSS, a base da web moderna.

Onde o JavaScript é utilizado?

  • Desenvolvimento Front-End: Para criar interfaces dinâmicas e interativas no navegador.
  • Desenvolvimento Back-End: Com Node.js, permitindo o uso de JavaScript no servidor.
  • Aplicações Mobile: Frameworks como React Native e Ionic permitem o desenvolvimento de aplicativos móveis.

JavaScript é uma linguagem interpretada ou compilada?

JavaScript é uma linguagem interpretada. O código é executado diretamente pelo navegador, linha por linha, sem um processo explícito de compilação.

Diferenças entre linguagem interpretada e compilada

  • Interpretada: Executa o código linha por linha diretamente no ambiente (JavaScript, Python).
  • Compilada: Converte o código em um arquivo executável antes da execução (C, Java).

Como o JavaScript pode modificar o comportamento de uma página web?

JavaScript pode modificar o comportamento de uma página manipulando o DOM, permitindo a alteração de conteúdo, estilos e reações a eventos de forma dinâmica, sem recarregar a página.

Variáveis no JavaScript

As palavras-chave usadas para declarar variáveis em JavaScript são:

  • var: Escopo de função e sofre "hoisting".
  • let: Escopo de bloco, introduzido no ES6.
  • const: Define uma constante, também com escopo de bloco.

O que são funções em JavaScript?

Funções são blocos de código reutilizáveis que podem ser chamados para executar tarefas. Elas podem receber parâmetros e retornar valores.

Exemplo de função:

function saudacao(nome) {
    return "Olá, " + nome + "!";
}
console.log(saudacao("João")); // Exibe: "Olá, João!"

Eventos em JavaScript

Eventos são ações ou ocorrências que acontecem no navegador e às quais podemos reagir com código JavaScript.

Exemplos de eventos:

  • click: Disparado quando um elemento é clicado.
  • submit: Disparado quando um formulário é enviado.

Exemplo de evento:

document.getElementById("meuBotao").addEventListener("click", function() {
    alert("Botão clicado!");
});

O que é o DOM (Document Object Model)?

O DOM é uma representação em árvore da estrutura HTML da página. Ele permite que o JavaScript manipule elementos HTML, estilos e estruturas.

Exemplo prático:

// Modifica o conteúdo de um elemento com o ID "titulo"
            document.getElementById("titulo").innerHTML = "Novo Título";

Bibliotecas e Frameworks em JavaScript

  • React: Biblioteca desenvolvida pelo Facebook para construir interfaces de usuário com componentes reutilizáveis.
  • Angular: Framework mantido pelo Google, usado para desenvolvimento de aplicações web robustas e escaláveis.
